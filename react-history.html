The History of React.js

From In-House Solution to Popular Web and Mobile Development Tool

React is a powerful and flexible open-source JavaScript library used for building user interfaces. It has become one of the most popular choices for frontend development due to its ease of use, high performance, and extensive ecosystem of tools and libraries. Since 2018, React has consistently received a higher percentage of monthly questions on Stack Overflow than any other comparable frontend framework, indicating strong levels of both usage and interest among developers.

Let's explore the process by which React gained adoption within Facebook, its eventual release to the public, and its meteoric rise to becoming one of the most popular and widely used frontend development libraries.

Major milestones and significant events in the history of React.js

  • 2011 - Development of React begins at Facebook by Jordan Walke who released an early prototype of React called "FaxJS". It was first deployed on Facebook's News Feed.
  • 2012 - Following its acquisition by Facebook, Instagram became the second product to adopt React in production. It was also the first application to be built end-to-end with React, after the library's successful use in production at Facebook.
  • 2013 - Jordan Walke introduced React to the world at JSConf US. React was released to the public as an open-source project.
  • 2014 - React Developer Tools became an extension of the Chrome Developer Tools. The release of React Hot Loader, which is a plugin that allows React components to be live reloaded without the loss of state.
  • 2015 - Netflix Likes React. Facebook released the first version of React Native, a mobile development framework that allows developers to use the React library to build native mobile applications.
  • 2016 - The release of React v15.0 introduced the concept of stateless functional components.
  • 2017 - Airbnb introduced their new open source library React Sketch.app
  • 2018 - React v16.3 was released, introducing several new features and improvements, including the highly-anticipated Context API.
  • 2019 - React 16.8 was released, introducing Hooks, which allow developers to use state and other React features in functional components without writing a class.
  • 2020 - React v17.0 was released, which introduces no new features but provides improvements to the developer experience and gradual upgrades.
  • 2021 - Facebook announced the release of React Server Components, a new way to write React components that can render on the server and stream updates to the client.
  • 2022 - The React Team announced React v18.0 which includes improvements like automatic batching, new APIs, streaming server-side rendering, and opt-in Concurrent React.
  • 2023 - React.js: The Documentary is available to watch for free on YouTube!
